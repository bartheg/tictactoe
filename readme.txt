"Tic Tac Toe" writen in Python 2.7 and PyGame 1.9.
It has sound effect, mouse and keyboard support.
No AI.

PL
Wymaga Python2 i Pygame
Myszą można klikać na pola i przycisk "Od nowa".

Klawisze:
Escape - wyłączenie gry
r - od nowa
0 (na klawiaturze numerycznej) - od nowa
1-9 (na klawiaturze numerycznej) - postawienie na polu O lub X

EN
Requirements Python2, Pygame.
Click to put O or X on the field. "Od nowa" restarts the game.
Escape - quit game
r - restart the game
0 (numeric keyboard) - restart the game
1-9 (numeric keyboard) - put O or X on the field
